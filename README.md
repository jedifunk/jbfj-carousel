jbfj-slider
=============

Wordpress content slider plugin.  Full responsive.

This plugin uses the great bxslider.js jQuery plugin by Steven Wanderski.
View the plugin on GitHub at https://github.com/wandoledzep/bxslider-4

Version: 1.2
